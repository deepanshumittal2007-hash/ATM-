# 💳 ATM Simulation System

A simple **command-line based ATM Simulation System** built using Python.
This project demonstrates a menu-driven program with proper **package architecture**, where users can perform basic banking operations like checking balance, depositing money, withdrawing money, and viewing transaction history.

---

## 🚀 Features

* 💰 Display Current Balance
* ➕ Deposit Money
* ➖ Withdraw Money
* 📄 View Transaction Statement
* 🔁 Menu-driven system using infinite loop

---

## 🛠️ Technologies Used

* 🐍 Python (Core Programming)
* 📦 Package Architecture (Modular Code Structure)

---

## 📁 Project Structure

```
ATM_Project/
│
├── main.py
│
├── data/
│   └── data.py
│
├── operations/
│   └── operations.py
```

---

## ⚙️ How It Works

* The program runs using a **menu-driven interface**.
* User selects an option from the menu.
* Functions from the `operations` module are called.
* Data like balance and transactions are stored in the `data` module.
